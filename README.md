# project-fototeca

## Ejercicio de una fototeca en la que puedes añadir imagenes y se muestran en un menú, las ordena por fecha

### Herramientas empleadas:
 - Node.js
 - Express
 - EJS
 - [Color-thief-node](https://www.npmjs.com/package/color-thief-node)
